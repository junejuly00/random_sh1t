## Image Processing & Metadata Standards

### EXIF Standards
The Exchangeable Image File Format (EXIF) standard is developed and maintained by:
- **CIPA (Camera & Imaging Products Association)**
- **JEITA (Japan Electronics and Information Technology Industries Association)**

EXIF metadata extraction and manipulation in this software follows publicly available specifications and standards. No proprietary algorithms or copyrighted material from camera manufacturers is used.

### IPTC Standards
The International Press Telecommunications Council (IPTC) metadata standards are publicly available specifications. This software implements standard IPTC metadata handling according to public documentation.

### XMP Standards
Adobe's Extensible Metadata Platform (XMP) is based on open standards:
- **RDF (Resource Description Framework)**: W3C standard
- **XML**: W3C standard
- **Dublin Core**: Open metadata standard

This software uses standard XML parsing techniques and does not include any Adobe proprietary code.
